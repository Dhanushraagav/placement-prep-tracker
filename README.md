# Placement Preparation Tracker

A JavaScript-based web application to track placement preparation topics.
Users can add, edit, complete, delete topics and monitor progress percentage.
Data is persisted using browser localStorage.

## Features
- Add preparation topics
- Mark topics as completed
- Edit topics
- Delete topics
- Filter (All / Completed / Pending)
- Progress percentage tracking
- Persistent data using localStorage

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla JS)

## How to Run
1. Clone the repository
2. Open `index.html` in any browser

## Learning Outcome
- DOM Manipulation
- Event Handling
- localStorage
- JavaScript CRUD operations
